Important FAQ

What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a tree-like structure that represents a webpage, allowing JavaScript to modify content, styles, and elements dynamically. It treats each part of the page as an object for easy interaction.

How do you select an element from DOM?

You can select an element from the DOM using methods like:
1. .getElementById("id")
2. .getElementsByClassName("className")
3. .getElementsByTagName("tagName")
4. .querySelector("selector")
5. .querySelectorAll("selector")

What is event delegation in the context of DOM, and why it is useful?

Event delegation is a method where one event listener is added to a parent element. This listener can handle events for all its child elements. When a child element triggers an event, it moves up to the parent, which can then respond to it. This technique helps boost performance by cutting down on the number of event listeners needed and makes code easier to manage, especially with elements that are added dynamically.

How do you manipulate an element's attributes and styles using the DOM?

You can change an element's attributes and styles with the DOM by getting the element using methods like document.getElementById() or document.querySelector(). To update attributes, use element.setAttribute('attributeName', 'value') or set it directly with element.attributeName = 'value'. For styles, adjust the style property, like element.style.propertyName = 'value'. This lets you change how elements look and act on a webpage.
